# **H1 Project Name**

**Luxury  Car   Showcase Website**

# **H2  Authors**

**Kelvin Kariuki**

## **H2 Description**

VELOCE is a modern, responsive luxury automotive showcase website designed to present high-performance and exotic vehicles in a premium digital showroom experience. The project focuses on sleek UI design, smooth user experience, and a high-end visual aesthetic inspired by elite automotive brands.

The website features a cinematic hero section, a curated vehicle showcase grid, and an elegant footer with social integration, showroom details, and newsletter subscription. Built using HTML, CSS, and Bootstrap, VELOCE demonstrates responsive design principles and modern front-end development techniques.

This project highlights attention to detail in layout structure, typography, hover effects, and visual hierarchy to create a luxury brand feel.

# **H1 Tech Stack**

*   **HTML**
*   **CSS**
*   **Bootstrap**
*   Google Fonts

# **H1 Key Features**

*   Fully responsive Bootstrap layout
*   Luxury-themed UI design
*   Hero section with strong branding focus
*   Car showcase grid with hover effects
*   Social media integration (Instagram, WhatsApp, etc.)
